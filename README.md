# SistemaLeilao
Teste Dev System
## Installation

Windows:

```sh
> Visual Studio: 
  Add-Migration CriacaoTabela -Context BancoContent 
  
> CLI Net: 
  dotnet ef migrations add CriacaoTabela
```
```sh
>  Visual Studio: 
  Update-Database
>  CLI Net: 
  dotnet ef database update

```
